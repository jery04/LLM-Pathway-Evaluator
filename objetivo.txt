1. ¿Qué demonios significa el tema 7?

El enunciado dice:

“Dado un conjunto de decisiones posibles en una carrera profesional, generar y evaluar múltiples trayectorias válidas bajo distintos criterios. El sistema debe utilizar un LLM para analizar y comparar cualitativamente las trayectorias generadas.”

Traducido a español normal:

Tú vas a construir un sistema que responda algo como:

“Quiero ser Data Scientist”
“Sé Python”
“Tengo poco tiempo”
“No me gusta matemática avanzada”

Y el sistema generará varias rutas profesionales posibles.

Ejemplo:

Ruta 1 — Rápida al empleo

Python → SQL → Data Analysis → Power BI → Junior Data Analyst

Ruta 2 — Más técnica

Python → Estadística → Machine Learning → Deep Learning → Data Scientist

Ruta 3 — Orientada a software

Python → Backend → APIs → MLOps → AI Engineer

Y después el sistema debe compararlas:

cuál es más rápida,
cuál requiere más esfuerzo,
cuál tiene más flexibilidad,
cuál da mejores oportunidades,
etc.

Y aquí entra el LLM (ChatGPT, Gemini, Llama, etc.):

El LLM no hace el algoritmo principal. El LLM ayuda a:

entender objetivos escritos en lenguaje natural,
interpretar preferencias del usuario,
comparar trayectorias cualitativamente.

Eso lo pide explícitamente el proyecto.

6. Cómo entra el LLM (MUY IMPORTANTE)

Esto es obligatorio en el proyecto.

Muchos estudiantes fallarán aquí porque harán un “adorno”.

Tú no.

El LLM debe ser funcional.

Uso 1 — Interpretar objetivos

Usuario escribe:

“Quiero trabajar en IA pero sin tanta matemática”

LLM traduce eso a:

{
 "goal": "AI Engineer",
 "avoid_math": true,
 "priority": "practical"
}
Uso 2 — Comparar rutas

El algoritmo genera:

Ruta A:

Python → SQL → Data Analysis

Ruta B:

Python → Statistics → ML

LLM responde:

La ruta A es más rápida para inserción laboral, mientras que la B ofrece una base técnica más sólida para roles avanzados en IA.

Eso es exactamente lo que pide el tema.

Uso 3 — Explicar recomendaciones

En vez de:

score=0.83

El usuario ve:

Se recomienda esta trayectoria porque aprovecha tus conocimientos previos en Python y minimiza prerequisitos matemáticos.

Eso queda súper profesional.
